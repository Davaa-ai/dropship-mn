# Bambi Room — Static & Non-Video Ad Playbook

Энэ баримт нь Reel/Видео контентоос **гадуур** ажиллах бүх төрлийн зураг, carousel, болон текст-д суурилсан реклам хувилбаруудыг нэгтгэсэн гарын авлага юм.

> **Яагаад энэ баримт хэрэгтэй вэ?**
> 2026 оны мэдээллээр static зураг реклам нь зарим тохиолдолд видеог давж performance өгч байгаа. Учир нь:
> * Canva дээр 10 минутад бэлэн болно → **тест хийх хурд** маш өндөр
> * 85% хүн дуугүйгээр scroll хийдэг → **текстэнд суурилсан мессеж** илүү сайн уншигдана
> * Retargeting funnel-д **static > video** (итгэл, spec, харьцуулалт)
> * Meta Andromeda update дараа static + video mix нь CPA бууруулдаг
>
> *(Fraser Cottrell, 2026: "500+ static тест хийсэн, энэ format-ууд л spend held")*

---

## ХЭСЭГ А: STATIC AD-ИЙН СЭТГЭЛ ЗҮЙН СУУРЬ

Static зураг рекламыг video шиг "эмоци → хувирал" arc-аар явуулж болохгүй. Оронд нь **нэг дэлгэцэнд нэг хүчтэй мессеж**-ийг аль болох хурдан шингээх зорилготой.

### Scroll зогсоох 3 зарчим (static-д зориулсан)

| # | Зарчим | Тайлбар | Бидний бүтээгдэхүүнд |
|---|--------|---------|---------------------|
| 1 | **Өнгө** | Хүчтэй, тод contrast бүхий block өнгө нь feed-д scroll зогсооно. Pretty дизайн биш, **анхаарал** чухал. | Бүтээгдэхүүний цэлгэг/шар/цагаан өнгийг хар/бараан фон дээр тавих |
| 2 | **Нэг тод мессеж** | Static-д 1 секундэд уншигдах гол headline хэрэгтэй. 2+ мессеж = skip. | "Нэг сандал. Гурван хэрэглээ." эсвэл "₮69,000-өөр 3 зүйлийг шийд." |
| 3 | **Social proof** | Бодит хүний review / тоо / баримт дээд хэсэгт байрлавал итгэл нэмнэ. | Ээжийн comment screenshot, эсвэл "75кг даац, BPA-free" spec |

## ХЭСЭГ Б: 8 БАТЛАГДСАН STATIC FORMAT

Доорх format бүрд:
- **Загвар тайлбар** (яах вэ)
- **Дизайны заавар** (Canva-д яаж хийх)
- **Монгол copy** (бэлэн текст)
- **Хаанд ашиглах** (TOF/MOF/BOF)

---

### FORMAT 1: "Асуудал → Шийдэл" нэг зураг (Problem-Solution Static)

**Сэтгэл зүй:** PAS (Problem-Agitate-Solution) framework. Асуудлаа мэддэг хүнд шийдлийг нэг зурагт багтааж харуулна.

**Дизайн:**
- Зурагны **дээд тал:** Асуудал → _"Хүүхдээ угаах болгонд хоёулаа зовдог уу?"_
- Зурагны **доод тал:** Шийдэл → Сандал дээр тухтай хэвтсэн хүүхдийн зураг + _"Тайвнаар угаагдана. ₮69,000."_
- Дунд нь нимгэн шугам эсвэл "→" тэмдэг
- Background: цэвэр нэг өнгө (цагаан эсвэл зөөлөн шар)

**Бэлэн copy (primary text):**
```
Хүүхдийнхээ үс угаах болгонд уйлж, эсэргүүцэж, ээж ядардаг уу?

Энэ 3 үйлдэлтэй сандал бүгдийг шийднэ:
✅ Хооллуулна
✅ Үс угаана
✅ Хөл угаана

PP материал. BPA-free. 75кг даац. 0-7 нас.
₮69,000 + үнэгүй хүргэлт.

👉 Дэлгэрэнгүйг линкээр
```

**Хаанд:** TOF (Top of Funnel) — шинэ хүмүүст

---

### FORMAT 2: "Зэрэгцээ харьцуулалт" (Side-by-Side / Price Anchoring)

**Сэтгэл зүй:** Anchoring effect. Хүмүүс зэрэгцүүлсэн тоог харахад хурдан, итгэлтэй шийдвэр гаргадаг.

**Дизайн:**
- Зурагны **зүүн тал:** хоолны сандал ~₮50k + угаалтын тавиур ~₮40k + суудал ~₮30k = **"₮120,000+"** (улаан, дамнуулсан зураг)
- Зурагны **баруун тал:** Манай нэг сандал = **"₮69,000"** (ногоон, checkmark)
- Доод хэсэг: _"Нэг авалт. Гурван хэрэглээ. Зай хэмнэнэ."_

**Бэлэн copy (primary text):**
```
Тус тусдаа 3 юм авбал: ₮120,000+ зарцуулж, гэр дүүрнэ.

Эсвэл яг ₮69,000-өөр:
→ Хоолны сандал ✓
→ Үс угаах тавиур ✓
→ Хөл угаах суудал ✓

Ганц юм. Зайгаа хэмнэ. Мөнгөө хэмнэ.
Үнэгүй хүргэлт.
```

**Хаанд:** TOF + MOF — логикоор ойлгуулах

---

### FORMAT 3: "Social Proof / Ээжийн сэтгэгдэл" (Fake Comment Card)

**Сэтгэл зүй:** Peer validation. Ээжүүд бусад ээж юу хэлснийг илүү итгэдэг.

**Дизайн:**
- Дээд хэсэг: Comment хэлбэрийн хайрцаг (avatar + нэр + review текст)
  - _"Анх хоолны сандал гэж авсан, хамгийн сайхан нь үс угаахад хэрэглэгдсэн. Хүүхэд огт уйлахаа больсон. Маш их баярлалаа 🙏"_ — **Сарнай Б.**
- Доод хэсэг: Бүтээгдэхүүний зураг + үнэ + CTA
- Фон: зөөлөн цайвар (cream/warm white)

**Бэлэн copy (primary text):**
```
Ээжүүд юу хэлж байна...

"Анх үс угаахад хэрэглэхээр авсан. Одоо хооллуулах, хөл угаахад ч ашигладаг. Гэрт зай бага эзлэхэд нь баярлалаа."

3 үйлдэлтэй сандал. ₮69,000.
Дэлгэрэнгүйг линкээр.
```

**Хаанд:** MOF + BOF — итгэл нэмж, шийдвэр гаргуулах

### FORMAT 4: "Collage / Grid" (4 талбарт хуваагдсан)

**Сэтгэл зүй:** Multiple visual entry points. 4 зүйл нэг зурагт харагдахад scroll зогсох магадлал нэмэгддэг.

**Дизайн (2×2 grid):**
| Зүүн дээд | Баруун дээд |
|-----------|------------|
| Бүтээгдэхүүний бүтэн зураг (цэвэр фон) | _"3 ХЭРЭГЛЭЭ. 1 САНДАЛ."_ (том үсгээр) |
| **Зүүн доод** | **Баруун доод** |
| 3 icon: 🍽️ Хоол · 🚿 Үс · 🦶 Хөл | _"₮69,000 · BPA-free · 0-7 нас"_ |

- Бүх grid нэг өнгөний scheme-тэй (тэнгэрийн хөх, эсвэл зөөлөн ягаан)
- Энгийн, цэвэрхэн, одоо ч ойлгомжтой

**Бэлэн copy (primary text):**
```
Нэг сандал, 3 хэрэглээ.

Хооллуулна 🍽️
Үс угаана 🚿
Хөл угаана 🦶

PP материал. BPA-free. 75кг даац.
₮69,000. Үнэгүй хүргэлт.
```

**Хаанд:** TOF — бүтээгдэхүүнийг танилцуулах

### FORMAT 5: "Organik Feed / Stories стиль" (Native-look Static)

**Сэтгэл зүй:** Camouflage effect. Реклам шиг харагдахгүй → scroll зогсоход илүү үр дүнтэй. 2026 оны хамгийн үр дүнтэй static format-уудын нэг (Fraser Cottrell).

**Дизайн:**
- Ээж хүүхдээ барьж дурангаас (selfie шиг) авсан зураг
- Instagram Stories-ийн font, sticker style-аар текст нэмэх
- _"Энэ юмыг олоод гайхсан 🤯"_
- _"Үс угаахдаа уйлахаа больсон"_
- _"₮69k + free delivery"_
- Ямар ч logo, brand element БАЙХГҮЙ (organic feed шиг харагдах)

**Бэлэн copy (primary text):**
```
Хүүхдэд минь яг хэрэгтэй зүйл олдсон ✨

Угаалтын өрөөнд ч, гал тогоонд ч ашиглагдаж байгаа.
Нэг юмаар 3 зүйлийг шийдсэн.

Линк bio-д.
```

**Хаанд:** TOF — Stories placement-д онцгой тохиромжтой

### FORMAT 6: "Headline Image" (Том гарчиг + Бүтээгдэхүүн)

**Сэтгэл зүй:** Bold headline нь мессежийг 0.5 секундэд шингээлгэнэ. Зураг headline-г тайлбарлах, дэмжих үүрэгтэй.

**Дизайн:**
- Дээд хэсэг (зургийн 60%): **"НЭМЭЛТ САНДАЛ АВАХ ХЭРЭГГҮЙ."** (том, хар/болд үсгээр)
- Доод хэсэг (зургийн 40%): Бүтээгдэхүүн 3 байрлалаар (хоол, үс, хөл) зэрэгцсэн зураг
- Хамгийн доор: _"₮69,000 · Үнэгүй хүргэлт · Линк bio-д"_
- Background: нэг пастель өнгө (шар, ягаан эсвэл цэнхэр)

**Headline хувилбарууд:**
1. _"НЭМЭЛТ САНДАЛ АВАХ ХЭРЭГГҮЙ."_
2. _"3 ЮМ АВАХ УУ, 1 ЮМ АВАХ УУ?"_
3. _"ҮС УГААХ СТРЕССЭЭ МАРТААРАЙ."_
4. _"ХЯМД ЮМ ҮНЭТЭЙ ТУСДАГ."_
5. _"₮69,000-ИЙН УХААЛАГ ХЭРЭГЛЭЭ."_

**Хаанд:** TOF + MOF — bold statement, ялангуяа warm audience-д

### FORMAT 7: "Spec Card" (Техникийн үзүүлэлтийн карт)

**Сэтгэл зүй:** Product-aware хүмүүст spec нь итгэл нэмнэ. "Би судлаад шийдсэн" гэсэн мэдрэмж.

**Дизайн:**
- Бүтээгдэхүүний зураг зүүн талд
- Баруун талд spec жагсаалт:
  - ✅ PP материал (BPA-free)
  - ✅ 0-7 нас хүргэл
  - ✅ 75кг хүртэл даац
  - ✅ 3 хэрэглээ: хоол, үс, хөл
  - ✅ Угсрахад 2 минут
  - ✅ Цэвэрлэхэд: нэг арчилт
- Доод хэсэг: ₮69,000 + CTA

**Бэлэн copy (primary text):**
```
Бүтээгдэхүүний мэдээлэл:

✅ PP материал — BPA-free, аюулгүй
✅ 0 сараас 7 нас хүртэл хэрэглэнэ
✅ 75кг хүртэл даац
✅ 3 хэрэглээ: хоол, үс угаах, хөл угаах
✅ Угсрахад 2 минут

₮69,000 + үнэгүй хүргэлт.
```

**Хаанд:** BOF (Bottom of Funnel) — retargeting, шийдвэр гаргах шатанд

**Сэтгэл зүй:** Price trivialization + Opportunity cost. Үнийг өдөр тутмын жижиг зардалтай харьцуулна.

**Дизайн:**

## ХЭСЭГ Г: COPY FRAMEWORK REFERENCE

Static ad дээрх primary text-ийг бичихэд ашиглах 3 framework:

### PAS (Problem → Agitate → Solution)
```
[Асуудал] Хүүхдийнхээ үс угаах болгонд стресстэй юу?
[Давамгайлах] Нуруу чилнэ. Хүүхэд уйлна. Ядаргаа нэмэгдэнэ.
[Шийдэл] Энэ сандал бүгдийг шийднэ. ₮69,000.
```

### BAB (Before → After → Bridge)
```
[Before] Өмнө нь: 3 өөр юм авч, гэр дүүрнэ, мөнгө дуусна.
[After] Одоо: Нэг юмаар 3 зүйлээ шийдсэн. Зай хэмнэнэ. Мөнгө хэмнэнэ.
[Bridge] 3 үйлдэлтэй хүүхдийн сандал. ₮69,000.
```

### AIDA (Attention → Interest → Desire → Action)
```
[A] ₮69,000-ийн ухаалаг хэрэглээ 👇
[I] Нэг сандалд 3 хэрэглээ: хоол, үс угаах, хөл угаах.
[D] PP (BPA-free). 75кг даац. 0-7 нас.
[A] Дэлгэрэнгүйг линкнээс үзээрэй.
```

## ХЭСЭГ Д: ДИЗАЙНЫ ЗАРЧМУУД

### Өнгө
- Бүтээгдэхүүний өнгөтэй contrast үүсгэх: бараан фон + цагаан/шар бүтээгдэхүүн
- Тод block color ашиглах ← scroll зогсооход #1 хүчин зүйл
- Гол мессеж дээр **хүчтэй contrast** байх (хар текст + цайвар фон, эсвэл цагаан текст + бараан фон)

### Текст
- Headline: **5 үг**-ээс ихгүй
- Sub-headline: нэг өгүүлбэр
- Дэлгэц дээр хэт олон текст бүү шах → "3 секундэд уншигдахгүй бол scroll дарна"
- Цэгцтэй hierarchy: headline → зураг → sub-text → CTA

### Зурагны шаардлага
- 1080×1080px (feed) эсвэл 1080×1920px (stories)
- Бодит бүтээгдэхүүний зураг хэрэглэх (stock биш)
- Хүүхэд + бүтээгдэхүүн хамт байгаа зургууд хамгийн сайн ажиллана
- Logo хэт том тавихгүй (итгэл бус, organic feel алдагдана)

## ХЭСЭГ Е: ТЕСТ ХИЙХ СТРАТЕГИ

### Static ad-ийг яаж тест хийх вэ

1. **Эхлээд 3-5 format** сонгож, тус бүрт 2-3 copy хувилбар бэлдэх
2. **ABO campaign** дотор тус тусдаа ad set-д байрлуулах
3. **3 өдөр** ажиллуулж, дараах metrics хянах:
   - **Hook rate:** Outbound CTR > 1% = сайн
   - **CPC:** ₮500-аас доош = сайн
   - **CPL (Cost per lead):** Landing page form submission дээр хэмжих
4. Хамгийн сайн ажиллаж байгаа 1-2 format-ыг **CBO campaign** руу зөөх

### Static + Video mix стратеги

Meta алгоритм 2026 онд **олон format-тай ad set**-ийг илүүд үздэг:
- 1 broad ad set дотор: **2 video (reel) + 2 static + 1 carousel** = хамгийн сайн mix
- Static нь retargeting-д маш хүчтэй → видеогоор анхаарлаа татсан хүнд static-аар итгэл/spec өгөх

## ХЭСЭГ Ж: PRODUCTION CHECKLIST (Static-д зориулсан)

Static зураг бэлдэхэд шаардлагатай asset-ууд:

- [ ] Бүтээгдэхүүний **цэвэр фон** дээрх зураг (3 өнцгөөс)
- [ ] Хүүхэд **хооллож** буй зураг
- [ ] Хүүхэд **үс угаалгаж** буй зураг
- [ ] Хүүхэд **хөл** угаалгаж / түр суусан зураг
- [ ] Ээж + хүүхэд хамт байгаа **selfie** стиль зураг (organic look)
- [ ] Ээжийн **тайвширсан** нүүрний зураг
- [ ] Бохирдсон tray-г **арчиж цэвэрлэсэн** before/after зураг
- [ ] Жижиг гал тогоонд **компакт** байршуулсан зураг
